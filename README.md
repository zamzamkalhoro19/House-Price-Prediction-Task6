# Task 6: House Price Prediction

## Objective
Predict median house prices in California districts based on features like income, location, and house characteristics using Linear Regression.

## Dataset
California Housing dataset containing 20,640 observations.

## Tools & Libraries
Python, Pandas, Scikit-learn, Matplotlib

## Model Used
Linear Regression

## Results
- Mean Absolute Error: ~$50,000
- R² Score: ~0.6
- Visualization: Actual vs Predicted prices scatter plot

## Insight
Linear Regression shows income and location are strong predictors of house prices. The model explains ~60% of price variance.
